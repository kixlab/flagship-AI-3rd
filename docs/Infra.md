## Tenserflow vs. Pytorch

|     | Tensorflow | Pytorch |
|:---|:-------------|:-------------|
| Code | a little complicated | simple |
| Graph | Tensorboard | No internal lib. (visdom?) |
| Debugging | TFDBG (more tough) | PDB |
| Community | Big | Medium, but growing |
| Area | Industrial, Research | Research |
